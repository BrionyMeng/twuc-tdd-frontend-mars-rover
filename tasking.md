火星车

- [ ]返回坐标和朝向
- [ ]面向不同方式时左转 Example：N->W; W->S
- [ ]面向不同方式时右转 Example：N->E; E->S
- [ ]前进
    - [ ]范围内能正常移动（happy path）：Example: (1,1,N) -> (1,2,N); (1,1,W) -> (0,1,W);
    - [ ]超出范围保持原地不动： Example: (0,0,S) -> (0,0,S)



控制中心

- 能够设置火星车初始位置
- 设置矩形区域大小
    - x,y >0 (Happy Path)
    - x<0 || y<0 抛出异常: "矩形区域坐标应该大于零" (Sad Path)
- 给火星车发送指令
